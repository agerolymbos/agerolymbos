Συχνές Ερωτήσεις

Πώς μπορώ να εγγραφώ στην HeroCard.gr;
Για να εγγραφείτε στην HeroCard.gr, πρέπει να συμπληρώσετε τη φόρμα εγγραφής στη σελίδα εγγραφής και να ακολουθήσετε τα βήματα που περιγράφονται εκεί.
Ποιες είναι οι απαιτήσεις για να γίνω μέλος;
Για να γίνετε μέλος της HeroCard.gr, πρέπει να είστε ηλικίας 18 ετών και άνω και να διαθέτετε μόνιμη διεύθυνση στην Ελλάδα.
Πώς μπορώ να επικοινωνήσω με την υποστήριξη;
Μπορείτε να επικοινωνήσετε με την υποστήριξη μέσω της φόρμας επικοινωνίας στην σελίδα Επικοινωνία ή μέσω email στο support@herocard.gr.

Συχνές Ερωτήσεις

Πώς μπορώ να ακυρώσω την εγγραφή μου;
Για να ακυρώσετε την εγγραφή σας, παρακαλώ επικοινωνήστε με την υποστήριξη μέσω της φόρμας επικοινωνίας ή μέσω email και θα επεξεργαστούμε το αίτημά σας.
Ποιες είναι οι διαθέσιμες προσφορές;
Οι διαθέσιμες προσφορές ποικίλλουν και περιλαμβάνουν εκπτώσεις σε καταστήματα, εστιατόρια, και υπηρεσίες. Επισκεφτείτε την ενότητα Προνόμια για να δείτε τις τρέχουσες προσφορές.
Μπορώ να χρησιμοποιήσω την κάρτα σε όλη την Ελλάδα;
Ναι, η HeroCard.gr μπορεί να χρησιμοποιηθεί σε επιχειρήσεις που συμμετέχουν στο πρόγραμμα σε όλη την Ελλάδα. Για να δείτε τις επιχειρήσεις που συμμετέχουν, επισκεφτείτε την ενότητα Συνεργαζόμενες Επιχειρήσεις.
Πώς μπορώ να προσθέσω μια επιχείρηση στην πλατφόρμα;
Για να προσθέσετε την επιχείρησή σας στην πλατφόρμα, επικοινωνήστε μαζί μας μέσω της φόρμας επικοινωνίας και δώστε μας τις απαραίτητες πληροφορίες για την επιχείρησή σας.

Συχνές Ερωτήσεις

Υπάρχει περιορισμός στον αριθμό των προσφορών που μπορώ να χρησιμοποιήσω;
Δεν υπάρχει περιορισμός στον αριθμό των προσφορών που μπορείτε να χρησιμοποιήσετε, αλλά ενδέχεται να ισχύουν όροι και προϋποθέσεις για κάθε προσφορά.
Πώς μπορώ να ενημερώσω τα προσωπικά μου στοιχεία;
Για να ενημερώσετε τα προσωπικά σας στοιχεία, συνδεθείτε στον λογαριασμό σας και επισκεφτείτε την ενότητα Ρυθμίσεις Λογαριασμού.
Πώς μπορώ να επαναφέρω τον κωδικό πρόσβασής μου;
Για να επαναφέρετε τον κωδικό πρόσβασής σας, κάντε κλικ στη σύνδεση "Ξεχάσατε τον κωδικό σας;" στη σελίδα σύνδεσης και ακολουθήστε τις οδηγίες.
Είναι ασφαλή τα προσωπικά μου δεδομένα;
Ναι, η HeroCard.gr δεσμεύεται για την προστασία των προσωπικών σας δεδομένων και τηρεί όλα τα απαραίτητα μέτρα ασφαλείας για την προστασία τους σύμφωνα με τη νομοθεσία περί προστασίας προσωπικών δεδομένων.
